# TP2 CUDA — Sum Reduction (Part 3)

CUDA implementation of the single-hidden-layer NN sum-reduction (Lab 2 Part 3).
Parts 1 & 2 (matrix multiplication, cuBLAS, TensorCore) are not implemented —
they are independent of Part 3 and not required by the submission template
(`../cuda files/CUDA_Results_summary.pdf`).

## Setup (one-time)

1. **CUDA Toolkit 13.2** must be installed at
   `C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v13.2`.
2. **VS BuildTools 2019** must be installed at
   `C:\Program Files (x86)\Microsoft Visual Studio\2019\BuildTools` (provides
   `cl.exe` and `vcvars64.bat` — needed by `nvcc` on Windows). The full VS IDE
   is **not** required.
3. VSCode extensions: `ms-vscode.cpptools`, `nvidia.nsight-vscode-edition`.

If `nvcc` rejects MSVC 14.29 with an "unsupported version" error, install
**VS Build Tools 2022** (https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2022)
and update the `vcvars64.bat` path in `build.bat` and `.vscode/tasks.json`.

## Build & run

From the `cuda_tp2/` folder in a normal `cmd` or `bash`:

```
build.bat hello        # smoke test
build\hello.exe

build.bat benchmark    # builds the full benchmark
build\benchmark.exe    # runs all kernel variants, prints results table
```

In VSCode: open `cuda_tp2/` → open any `.cu` → `Ctrl+Shift+B` runs the
"nvcc: build current file" task.

## Layout

```
src/
  hello.cu           # toolchain smoke test
  common.cuh         # data generation, CPU reference, timing macros
  reductions.cuh    # all kernel variants (naïve + 4 optimizations)
  nn_naive.cu        # full NN pipeline using naïve kernel (Part 3-1)
  nn_optimized.cu    # full NN pipeline using best kernel (Part 3-2)
  benchmark.cu       # head-to-head comparison + results table
results.md           # benchmark output for the report
```
